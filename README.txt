Задание:
-Проект на ASP.NET Web API.
-Должен по внешнему запросу создавать короткую ссылку на URL.
-При попытке перейти по созданной короткой ссылке должен
перенаправлять по сохраненному URL.
-10.000 коротких ссылок в день создается.
-Ссылка хранится в среднем 10 лет.
-100.000 обращений на редирект по коротким ссылкам в день.
-Периодически нужно чистить БД от устаревших ссылок.
-При попытке создать ссылку на уже существующий элемент в БД нужно
возвращать уже существующий элемент, а не генерировать новый.
